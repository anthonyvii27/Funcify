## Client

### Environments Variables

| Parameter                    | Description                             |
|:-----------------------------|:----------------------------------------|
| `ENVIRONMENT`                | `example1` or `example2` (**Required**) |

### Run Locally

Instructions about how to run the client

1. Ensure that you're in the client directory

```shell
cd client
```

2. Install dependencies

```shell
npm install

# or

npm i
```

3. Start the project

```shell
npm run start:dev
```
